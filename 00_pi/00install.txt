Configuration
-------------
*Connect WIFI
*Run raspi-config
	*Change password
	*Change hostname
	*Disable boot splash
	*Enable SSH & VNC
	*Audio Force Headphone
	*Expand Filesystem
	*Localisation - Keyboard - Generic 104-Key - Other/English (US)
	*Reboot
*Setup screen/background
	*Add CPU and Minimize to panel
*Setup Audio defaults
*cp /usr/share/zoneinfo/America/New_York /etc/localtime 
*Copy config.txt to /boot for small panel
*apt-get install xscreensaver
