TARDIS Pi Install
----+----+----+----+----+

Configuration (2022)
-------------

*First Boot
   *Set Country
   *Setup user/password
   *Setup WiFi
*Second Boot
   git clone https://github.com/TriangleTARDIS/TARDIS-Pi.git ~/TARDIS-Pi/
   sudo env HNAME=hostName sh ~/TARDIS-Pi/00_pi/raspi-config.txt

   pip3 install -r ~/TARDIS-Pi/requirements.txt
   cp -rv ~/TARDIS-Pi/00_pi/home/. ~/
   sed -i'' "s/~/\/home\/$USER/" ~/Desktop/TARDIS-*.desktop

   *Setup a screensaver
   *Configure audio if necessary
   *Run Calibration to verify LEDs

   sudo shutdown -r now

*Append config.txt.panel to /boot for small panel (800x480)



Configuration (Deprecated/Manual)
-------------
*SD Card /boot/
   *Copy 00_pi/raspi-config.txt
   *Create empty ssh.txt
*First Boot, SSH or console to host
   sudo env HNAME=hostName WNAME=wifiName WPWD=wifiPassword sh /boot/raspi-config.txt
*Second Boot, see above...
